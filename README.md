# Hikiverse 🌌

A modern and responsive anime catalog landing page. This project serves as a practical showcase of React 19, TypeScript, and various UI/animation libraries.

## 🚀 Key Features

- **Interactive UI:** Smooth animations powered by `framer-motion`.
- **Modern Design:** Built with `Ant Design` and `antd-style`.
- **Responsive Layout:** Fully adaptive design for a seamless experience on all devices.
- **High Performance:** Lightning-fast builds using `Vite`.

## 🛠 Tech Stack

- **Frontend:** React 19, TypeScript
- **UI Framework:** Ant Design, antd-style
- **Animations:** Framer Motion, Keen Slider
- **Icons:** Lucide React
- **Build Tool:** Vite

## 📦 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Cartnix/HikiVerse
   cd hikiverse
   ```
2. **Install dependences:**

```bash
    npm install
```

3. **Run in development mode:**

```bash
    npm run dev
```

4. **Build for production**

```bash
    npm run build
```
